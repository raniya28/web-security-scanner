# Web Application Security Scanner (OWASP Top 10)

## Description
A Python-based web application security scanner that detects common OWASP Top 10 vulnerabilities such as Cross-Site Scripting (XSS) and SQL Injection. The project uses a deliberately vulnerable Flask web application for ethical and controlled security testing.

## Features
- Reflected XSS detection
- Basic SQL Injection detection
- Local vulnerable test application
- Ethical scanning approach

## Technologies Used
- Python 3
- Flask
- Requests
- HTML (Jinja2)
